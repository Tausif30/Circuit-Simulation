<H1>
  Circuit Simulations
</H1>
<H3>
  A Repository Compromising of Simulations for Different Circuits, needed for a basic foundation. 
  <br>
  Contains both Analogue and Digital Circuits. 
  <br>
  Simulation for Analogue Circuits can be run in LTSpice or other SPICE Softwares.
  <br>
  Simulation for Digital Circuits can be run in LogiSim or other similar Softwares.
</H3>
