<h1 align="center">Circuit Simulations</h1>

<p align="center">
A repository containing <b>analog</b> and <b>digital circuit simulations</b> for building a strong foundation in electronics and circuit design.
</p>

---

<h2>Software</h2>

<h3>Analog Circuits</h3>
<p>
Analog simulations are created using <b>LTSpice</b> and can be opened with most SPICE-based simulators.
</p>

<ul>
  <li>
    <a href="https://www.analog.com/en/design-center/design-tools-and-calculators/ltspice-simulator.html">
      LTSpice
    </a>
  </li>
</ul>

<h3>Digital Circuits</h3>
<p>
Digital simulations are created using <b>Logisim Evolution</b> and compatible digital logic simulators.
</p>

<ul>
  <li>
    <a href="https://github.com/logisim-evolution/logisim-evolution">
      Logisim Evolution
    </a>
  </li>
</ul>

---

<h2>Repository Structure</h2>

<h3>Analog</h3>
<ul>
  <li>Basic Electronic Circuits</li>
  <li>Differential Amplifiers</li>
  <li>Operational Amplifier Circuits</li>
  <li>Amplifiers and Feedback Topologies</li>
  <li>Filters and Oscillators</li>
  <li>PWM and Switching Circuits</li>
  <li>Power Electronics and SMPS</li>
  <li>Voltage Regulation</li>
</ul>

<h3>Digital</h3>
<ul>
  <li>Logic Gates</li>
  <li>Multiplexers and Demultiplexers</li>
  <li>Arithmetic Circuits</li>
  <li>Display Circuits</li>
</ul>

---

<h2>Getting Started</h2>

<p>
Clone the repository:
</p>

```bash
git clone https://github.com/<username>/Circuit-Simulations.git
```

<p>
Open <b>.asc</b> files using LTSpice or another SPICE simulator, and <b>.circ</b> files using Logisim Evolution.
</p>
